# Synergy - USAF DARPA Contract Work 1998-2007

## Context

Nine-year independent contractor engagement via Sunetos Inc to Synergy (later ICFI) as prime for USAF and DARPA contracts. Transitioned out of Synergy's employee role in 1998 and continued the same logistics research lineage as an independent contractor, adding enterprise Java and distributed architecture work as the stack evolved through the early 2000s.

## Role

Architect and lead developer for a sequence of multi-tier distributed component systems delivered via Synergy/ICFI to USAF and DARPA. Designed server layers on EJB/JRun and JBoss, built CORBA integration bridges to legacy Win32 applications, and mentored junior Java developers.

## Achievements

- System architect and lead developer for **WRM-CA** (War Reserve Materiel Capability Assessment, 1999-2001) — multi-tier distributed Java Swing system on Allaire's EJB server (JRun) with Oracle persistence and JMS notification. CORBA-accessible server components for non-Java client integration. #architecture #j2ee #ejb #corba #defense
- CORBA and Java developer for **ALA** (Air Logistics Application, 1998-1999 for DARPA) — added CORBA interface to an existing Delphi Win32 application to serve as a server component in the Java/CORBA system. #corba #delphi #java #darpa
- CORBA developer for **CBP** (Collaborative Beddown Planner, 1999-2000) — added CORBA interface to existing Delphi Win32 application as a server component in the EJB system. #corba #delphi #defense
- System architect and developer for **UTC-DT** (Unit Type Code Development and Tailoring) — custom peer-to-peer socket protocol for multi-user distributed equipment requirements tailoring (continuation of employee-era work). #peer-to-peer #distributed #defense
- System analysis for a **multi-tier DII-COE system** (2001) — analyzed existing systems to develop requirements and level of effort for a Defense Information Infrastructure Common Operating Environment-compliant logistical analysis tool for the Air Force. #systems-analysis #dii-coe #defense
- Designed and developed the Java subsystems for a **DARPA software agent research prototype** (2004-2005) — rule engine, end-user rule builder GUI, tracking subsystem, system utilities. Mentored team members in J2EE design, Java patterns, and refactoring. #darpa #agents #mentoring #rule-engine
- Additional engagements through 2007 included enterprise workflow, WebSphere and WebLogic migration work, and semantic web research (OWL, Protégé) in humanitarian aid domains. #enterprise #j2ee #semantic-web

## Tech Stack

Java (JBuilder then IntelliJ IDEA), J2EE (Allaire/JRun, JBoss/Jetty, IBM WebSphere, BEA WebLogic), Delphi (client/server legacy integration), CORBA (Inprise/VisiBroker), EJB 1.1, Oracle, Palm OS J2ME, PocketPC/WinCE Personal Java, Swing, JMS.

## Private Notes

_Billed via Sunetos Inc to Synergy/ICFI prime. End clients were USAF Armstrong Labs/AFRL for most contracts, DARPA for ALA and Software Agent Research. Transition out of this arrangement was gradual through 2007 — it diluted as civilian Java and iOS work took over rather than ending sharply. The specific end year 2007 is the last resume-documented project on this lineage; informal relationship may have continued past that._
